# 🎬 Cinema System Reservation

Backend REST API for a cinema seat reservation system.

## Tech stack
- Java 21
- Spring Boot
- Spring Web (REST)
- Spring Data JPA
- PostgreSQL
- Maven
- JUnit

## Features
- Create screening reservations
- Prevent double booking (DB constraint)
- Cancel reservations
- List reserved seats for a screening
